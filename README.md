# Demon Cat Tamagotchi 😈🐱

Веб-версия виртуального питомца в жанре тамагочи, главный герой которого — милый **красный демон-кот**: кошачье тело, изогнутые демонические рожки вместо ушей. Игрок заботится о питомце, поддерживая его показатели в хорошем состоянии, и может рисовать вместе с ним в встроенном скетчбуке.

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- Node.js (опционально, для фронтенда)

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/sindovna442/shiny-fiesta.git
cd shiny-fiesta
```

2. **Установите зависимости Python:**
```bash
cd backend
pip install -r requirements.txt
```

3. **Запустите Flask сервер:**
```bash
python app.py
```

4. **Откройте в браузере:**
```
http://localhost:5000
```

## 🎮 Основные механики

### Уход за питомцем
- 🍖 **Кормление** — восполняет голод
- 🤲 **Поглаживание** — поднимает настроение
- 🛁 **Мытьё** — поддерживает чистоту
- 💤 **Сон** — восстанавливает энергию

### Показатели питомца
- **Голод** (0-100) — растёт со временем
- **Чистота** (0-100) — снижается со временем
- **Настроение** (0-100) — зависит от заботы
- **Энергия** (0-100) — восстанавливается во время сна
- **Здоровье** (0-100) — зависит от других показателей

### Скетчбук
- Встроенный редактор рисования
- Инструменты: кисть, ластик, палитра цветов
- История действий: отмена/повтор
- Сохранение рисунков
- Показ рисунков питомцу повышает его настроение

## 🔧 API Endpoints

### Питомец
- `POST /api/pet/create` — Создать нового питомца
- `GET /api/pet/<pet_id>` — Получить статус питомца
- `POST /api/pet/<pet_id>/feed` — Накормить питомца
- `POST /api/pet/<pet_id>/pet` — Погладить питомца
- `POST /api/pet/<pet_id>/wash` — Помыть питомца
- `POST /api/pet/<pet_id>/sleep` — Уложить питомца спать
- `POST /api/pet/<pet_id>/view-drawing` — Показать рисунок питомцу

### Рисунки
- `GET /api/sketches/<pet_id>` — Получить все рисунки
- `GET /api/sketches/<pet_id>/<sketch_id>` — Получить конкретный рисунок
- `POST /api/sketches/<pet_id>/save` — Сохранить новый рисунок
- `DELETE /api/sketches/<pet_id>/<sketch_id>` — Удалить рисунок

## 📝 Примеры использования

### Создание питомца
```bash
curl -X POST http://localhost:5000/api/pet/create \
  -H "Content-Type: application/json" \
  -d '{"name": "Демонический Кот"}'
```

### Кормление питомца
```bash
curl -X POST http://localhost:5000/api/pet/{pet_id}/feed
```

### Сохранение рисунка
```bash
curl -X POST http://localhost:5000/api/sketches/{pet_id}/save \
  -H "Content-Type: application/json" \
  -d '{"title": "Мой рисунок", "imageData": "data:image/png;base64,..."}'
```
