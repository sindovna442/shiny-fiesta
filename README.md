# shiny-fiesta
Веб-версия виртуального питомца в жанре тамагочи, главный герой которого — милый **красный демон-кот**: кошачье тело, изогнутые демонические рожки вместо ушей. Игрок заботится о питомце, поддерживая его показатели в хорошем состоянии, и может рисовать вместе с ним в встроенном скетчбуке — цифровом блокноте с простым редактором изображений.
