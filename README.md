# Demon Cat Tamagotchi 😈🐱

Веб-версия виртуального питомца в жанре тамагочи с коллекцией мини-игр. Главный герой — милый **красный демон-кот**: кошачье тело, изогнутые демонические рожки вместо ушей. Игрок заботится о питомце и играет в 6 встроенных мини-игр.

---

## 🎮 Мини-игры

### 🏄 Морской Сёрф
Собери монеты, уклоняйся от препятствий и прыгай через рыб на серф-доске по волнам.
- **Управление:** свайп влево/вправо для смены полосы, свайп вверх для прыжка
- **Монеты:** 🦪 ракушки — собирай для очков (+5 за прыжок через рыбу)
- **Game Over:** столкновение с препятствием (брёвна, скалы, лодки)

### 🍳 Шеф-Повар
Расслабляющая игра-конструктор без таймера: собирай блюдо из ингредиентов и корми питомца.
- **3 основы:** 🍕 Пицца, 🍔 Бургер, 🎂 Торт
- **Этапы для пиццы:** соус → начинка → запекание (анимация!)
- **Этапы для бургера:** котлета → слои → соус → верхняя булочка
- **Этапы для торта:** глазурь → крем → топпинги
- **Кормление:** перетаскивай кусочки питомцу в рот
- **Реакции питомца:** огненное дыхание (острое), раздувшийся животик (большой бургер), сердечки (сладкий торт)
- 🎵 Звуковые эффекты: шипение запекания, чавканье, довольный писк

### 🔥 Hellfire Balls
Разрушай разноцветные блоки огненными шарами — аркадная головоломка.
- **Цель:** уничтожить все блоки на поле за минимальное количество выстрелов
- **Управление:** наведи курсор на цель → клик для выстрела
- **Размер блоков:** увеличен для удобства (95% ячейки)
- **Траектория:** анимированный след полёта шара

### ♟️ Шахматы
Игра против AI с кошачьими фигурами.
- **Фигуры:** ♟ пешки, ♞ кони, ♝ слоны, ♛ ферзь, ♚ король, ♜ ладьи
- **AI противник:** минимакс с альфа-бета отсечением
- **Уровни:** Easy (глубина 1), Medium (глубина 2), Hard (глубина 3)
- **Клетки:** подсветка выбранной фигуры и доступных ходов

### 🔢 Судоку
Классическая головоломка с генерацией новых досок.
- **Генерация:** каждая игра уникальна
- **Проверка:** автоматическая при заполнении
- **Подсказки:** предзаполненные клетки не редактируются

### 🎨 Скетчбук
Полноценный редактор рисования с сохранением в галерею.
- **Инструменты:** 🖌️ Кисть, 🧹 Ластик, 🪣 Заливка
- **Цвета:** палитра из 12 цветов
- **Размер кисти:** регулируемый
- **История:** ↩️ Отмена (Ctrl+Z), ↪️ Повтор (Ctrl+Y)
- **Сохранение:** в галерею через API + localStorage backup
- **Показ питомцу:** рисунки можно показывать коту для поднятия настроения

### 🫧 Антистресс
Лопание пузырьков со звуковыми эффектами для релаксации.

## 🚀 Быстрый старт

### Требования
- Python 3.8+

### Установка и запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/sindovna442/shiny-fiesta.git
cd shiny-fiesta

# Установите зависимости
make install
# или: cd backend && pip install -r requirements.txt

# Запустите сервер
make dev
# или: cd backend && python app.py
```

Откройте в браузере: **http://localhost:5000**

> **Makefile** автоматизирует установку (`make install`) и запуск (`make dev`).

## 🎮 Основные механики питомца

### Уход за питомцем
- 🍖 **Кормление** — восполняет голод
- 🤲 **Поглаживание** — поднимает настроение
- 🛁 **Мытьё** — поддерживает чистоту (кнопка в ванной)
- 💤 **Сон** — восстанавливает энергию (кровать)
- 🖼️ **Показать рисунок** — улучшает настроение

### Показатели питомца
- **Голод** (0-100) — растёт со временем
- **Чистота** (0-100) — снижается со временем
- **Настроение** (0-100) — зависит от заботы
- **Энергия** (0-100) — восстанавливается во время сна
- **Здоровье** (0-100) — зависит от других показателей

### Сохранение
- 💾 **Автосохранение** — питомец сохраняется в `localStorage` после каждого действия
- 🔇 **Mute-кнопка** — выключение всех звуков (сохраняется в `localStorage`)

## 🔧 API Endpoints

### Питомец
| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/pet/create` | Создать нового питомца |
| `GET` | `/api/pet/<pet_id>` | Получить статус питомца |
| `POST` | `/api/pet/<pet_id>/feed` | Накормить питомца |
| `POST` | `/api/pet/<pet_id>/eat` | Съесть еду (уменьшает голод) |
| `POST` | `/api/pet/<pet_id>/pet` | Погладить питомца |
| `POST` | `/api/pet/<pet_id>/bath-toggle` | Переключить ванну |
| `POST` | `/api/pet/<pet_id>/bed-toggle` | Переключить кровать |
| `POST` | `/api/pet/<pet_id>/view-drawing` | Показать рисунок питомцу |

### Рисунки
| Метод | Путь | Описание |
|---|---|---|
| `GET` | `/api/sketches/<pet_id>` | Получить все рисунки |
| `GET` | `/api/sketches/<pet_id>/<sketch_id>` | Получить конкретный рисунок |
| `POST` | `/api/sketches/<pet_id>/save` | Сохранить новый рисунок |
| `DELETE` | `/api/sketches/<pet_id>/<sketch_id>` | Удалить рисунок |

## 📝 Примеры использования

```bash
# Создать питомца
curl -X POST http://localhost:5000/api/pet/create \
  -H "Content-Type: application/json" \
  -d '{"name": "Демонический Кот"}'

# Накормить
curl -X POST http://localhost:5000/api/pet/{pet_id}/feed

# Переключить ванну
curl -X POST http://localhost:5000/api/pet/{pet_id}/bath-toggle

# Сохранить рисунок
curl -X POST http://localhost:5000/api/sketches/{pet_id}/save \
  -H "Content-Type: application/json" \
  -d '{"title": "Мой рисунок", "imageData": "data:image/png;base64,..."}'
```

## 🎨 Технологии

### Backend
- **Python 3** + **Flask** — REST API сервер
- **Flask-CORS** — поддержка кросс-доменных запросов

### Frontend
- **HTML5** + **CSS3** — градиенты, анимации, кастомные скроллбары
- **JavaScript (Vanilla)** — 6000+ строк кода
- **Canvas API** — питомец, анимации, все мини-игры
- **Drag-and-Drop API** — перетаскивание ингредиентов в ChefGame
- **Web Audio API** — звуковые эффекты

### Хранение
- **In-memory** (серверная часть) — для демо
- **localStorage** (браузер) — резервное копирование питомца и настроек
